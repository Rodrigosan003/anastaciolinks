# 🔗 Links

Uma página de links simples e responsiva para centralizar minhas redes sociais e formas de contato.

## 📖 Sobre

Projeto desenvolvido para reunir em um único lugar meus principais links e canais de comunicação, utilizando uma interface minimalista e moderna.

## ✨ Funcionalidades

* Foto de perfil personalizada.
* Links para redes sociais e contato.
* Layout responsivo.
* Efeito de fundo com blur.
* Animações suaves ao passar o mouse.
* Interface simples e elegante.

## ⚙️ Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/Rodrigosan003/anastaciolinks
```

2. Acesse a pasta do projeto:

```bash
cd seu-repositorio
```

3. Abra o arquivo `index.html` no navegador ou utilize uma extensão como **Live Server** no VS Code.
